# Grace Delacroix

This website presents selected quantitative work from my undergraduate studies in political science. The projects apply statistical analysis to questions of development, democratic representation, and political communication. Together they reflect my ability to use data to examine how structural conditions shape social and political outcomes.

The site is built using R, R Markdown, and the Distill framework, with interactive visualisations powered by Plotly.

## Featured Projects

**Impact of Mother's Characteristics on Child Test Scores**

This project investigates how maternal IQ and educational attainment relate to child cognitive outcomes using NLSY data. The analysis explores how family advantage operates not only through baseline ability but through structural stability, showing that educational attainment strengthens the translation of maternal resources into child development.

**Democratic Scale: UK Constituency Size Analysis**

This study examines variation in constituency sizes across the United Kingdom to understand patterns in democratic representation. By comparing voter distributions across the countries of the UK, the analysis highlights structural imbalances that shape electoral equality and representation.

**Political Communications: A/B Testing and Treatment Effects**

This project evaluates a digital A/B test to determine how different homepage messaging strategies influence supporter sign-up rates. Using treatment effect estimation and interaction modelling, the analysis explores how political communication effectiveness varies across demographic groups.

**Partisan Signals in Defeat: Text Analysis of Concession Speeches**

This project applies Naive Bayes text classification to US presidential concession speeches to detect partisan language differences between Democrats and Republicans at the paragraph level. Even in moments of political defeat, partisan rhetorical habits persist.

## About the Website

The portfolio documents my progression in quantitative analysis within political science. Each project combines statistical modelling, data visualisation, and substantive interpretation to examine how institutional structures and social conditions influence outcomes.
